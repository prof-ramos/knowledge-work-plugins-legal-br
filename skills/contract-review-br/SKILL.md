# Skill — Contract Review (Brasil)

## Princípios
- Não é aconselhamento jurídico.
- Sempre explicitar premissas (papel das partes, tipo de contrato, se há dados pessoais).

## Cláusulas mínimas a cobrir (Brasil)
- Objeto/escopo
- Preço/reajuste/tributos
- Prazo/renovação/rescisão/multa
- Responsabilidade/limitação/exclusões
- Indenização
- Confidencialidade
- Propriedade intelectual
- LGPD (se aplicável)
- Foro/arbitragem

## Classificação
- VERDE: aceitável
- AMARELO: negociar
- VERMELHO: escalar

## Formato de saída
- Resumo executivo
- Top issues
- Tabela/itens por cláusula com: trecho, risco, sugestão
